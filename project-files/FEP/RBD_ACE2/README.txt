This folder containes:
1- molecular dynamics parameters for GROMACS 2020 software (.mdp files)
2- the folder named "native_RBD_ACE2" contains all input files for free energy simulation of native RBD-ACE2 complex.
3- the folder named "oxidized_RBD_ACE2" contains all input files for free energy simulation of oxidized RBD-ACE2 complex.

Any sub-folders has its own README file.
