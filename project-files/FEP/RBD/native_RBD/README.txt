This folder containes:
1- the folder named "restraints" contains the position restraints of proteins.
2- the folder named "toppar" contains the bonded and non-bonded parameters of atoms in charmm36 force field.
3- the topology file named "topol_native_RBD". 
4- the index file named "index_native_RBD" contains some user definable sets of atoms.
5- the configuration files named "replica_1", "replica_2", "replica_3", and "replica_4" contains the coordinates of all atoms of system, including RBD, water molecules and ions.

Any sub-folders has its own README file.
