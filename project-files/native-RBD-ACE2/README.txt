This folder containes:
1- the folder named "restraints-RBD-ACE2" contains the position restraints of proteins.
2- the folder named "toppar-RBD-ACE2" contains the bonded and non-bonded parameters of atoms in charmm36 force field.
3- the topology file named "topol-RBD-ACE2". 
4- the index file named "index-RBD-ACE2" contains some user definable sets of atoms.
5- the configuration file named "conf-RBD-ACE2" contains the coordinates of all atoms of system, including RBD, ACE2, water molecules and ions.

Any sub-folders has its own README file.
