This folder containes:
1- the folder named "restraints-Ox-RBD-GRP78" contains the position restraints of proteins.
2- the folder named "toppar-Ox-RBD-GRP78" contains the bonded and non-bonded parameters of atoms in charmm36 force field.
3- the topology file named "topol-Ox-RBD-GRP78". 
4- the index file named "index-Ox-RBD-GRP78" contains some user definable sets of atoms.
5- the configuration file named "conf-Ox-RBD-GRP78" contains the coordinates of all atoms of system, including oxidized RBD, GRP78, water molecules and ions.

Any sub-folders has its own README file.
